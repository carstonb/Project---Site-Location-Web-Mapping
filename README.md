# Omnisharp Site Location Web Mapping

This project is a web-based mapping tool for visualizing the locations of various plant sites and headquarters of Omnisharp LLC. The map is built using Leaflet and includes features such as locating the user's position, resetting the map view, and displaying distances between sites.

## Features

- **Interactive Map**: Displays plant sites and headquarters on an interactive map.
- **Custom Icons**: Uses custom icons for different types of sites (beef, pork, poultry, HQ).
- **Distance Calculation**: Calculates and displays distances between sites.
- **Locate Button**: Allows users to find their current location on the map.
- **Reset Button**: Resets the map view to the initial state.
- **Toggle Buttons**: Includes buttons to toggle connections between Mexico and USA plants, show road distances, and include/exclude HQ in distance calculations.

## Installation

1. Clone the repository to your local machine.
   ```sh
   git clone <repository-url>
   ```
2. Navigate to the project directory.
   ```sh
   cd Project - Site Location Web Mapping
   ```

## Usage

1. Open the `index.html` file in your web browser.
2. Use the interactive map to explore the plant sites and headquarters.
3. Click the "Locate" button to find your current location on the map.
4. Click the "Reset" button to reset the map view to the initial state.
5. Use the toggle buttons to connect Mexico to USA plants, show road distances, and include/exclude HQ in distance calculations.

## Files

- `index.html`: The main HTML file that contains the structure of the web page.
- `style.css`: The CSS file that contains the styles for the web page.
- `script.js`: The JavaScript file that contains the logic for the interactive map.
- `Images/`: A directory containing custom icons for the map.

## Dependencies

- [Leaflet](https://leafletjs.com/): A JavaScript library for interactive maps.
- [Leaflet Routing Machine](https://www.liedman.net/leaflet-routing-machine/): A routing control for Leaflet.
